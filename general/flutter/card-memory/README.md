# Card Memory

Card Memory Game App implemented on Flutter.

## Instructions

To start this app:

- Installing Dependencies

```
flutter pub get
```

- Running Flutter Application:

```
flutter run
```

- Run Flutter Test Application:

```
flutter test
```

- Run Linter:

```
flutter analyze
```

- Run Formatter:

```
flutter format .
```
