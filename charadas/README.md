# Charadas

Charadas App implemented on Flutter.

## Instructions

To start this app:

- Installing Dependencies

```
flutter pub get
```

- Running Flutter Application:

```
flutter run
```

- Run Flutter Test Application:

```
flutter test
```

- Run Linter:

```
flutter analyze
```

- Run Formatter:

```
flutter format .
```

## Basic Features

- User can see a display showing a button containing "Start".
- User can click the 'Start' button to display:
    - The movie to describe.
    - A countdown that start with 30 seconds.
- User can heard a sound when the countdown goes to 0.
- After the sound the display return to the main page.
